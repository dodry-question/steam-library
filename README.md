# Steam Library Manager

Веб-приложение для управления и анализа библиотеки Steam с персонализированными рекомендациями игр на основе искусственного интеллекта.

## Основные возможности

Приложение позволяет просматривать библиотеку Steam с актуальными ценами и скидками, получать персонализированные рекомендации на основе анализа профиля, искать и сортировать игры по различным параметрам. Реализован адаптивный дизайн для desktop и mobile устройств, есть возможность просмотра библиотек других пользователей и статистика времени в играх. Добавлена поддержка темной и светлой темы с кэшированием данных для быстрой загрузки.

## Технологии

Backend построен на FastAPI с использованием SQLModel для работы с базой данных SQLite. Для асинхронных HTTP запросов используется httpx. Frontend реализован на чистом JavaScript с модульной архитектурой, стили написаны на CSS3 с поддержкой адаптивного дизайна и тем. Шаблоны генерируются через Jinja2.

Приложение интегрируется с несколькими внешними API: Steam Web API для получения данных о библиотеке, Steam Store API для информации о ценах и жанрах, VseGPT API (GPT-4o-mini) для генерации рекомендаций.

## Как это работает

Приложение использует SQLite для кэширования данных о играх. При первом запросе данные загружаются из Steam API и сохраняются в локальную базу. При повторных запросах информация берется из кэша, который автоматически обновляется с учетом времени обновления Steam Store (10:10 по тихоокеанскому времени).

Реализована система потоковой загрузки данных: сначала отображается список игр с базовой информацией, затем детальные данные (цены, жанры, скидки) загружаются в фоновом режиме, и карточки игр обновляются по мере получения данных. Используется rate limiting для предотвращения блокировки Steam API.

Рекомендации генерируются с помощью языковой модели GPT-4o-mini. В глобальном режиме анализируются топ-15 игр по времени в игре с учетом выбранного настроения, при этом исключаются игры, уже имеющиеся в библиотеке. В режиме выделения пользователь может выбрать конкретные игры (Shift+клик или долгое нажатие на мобильных), и модель ищет похожие по геймплею и атмосфере. История рекомендаций сохраняется в рамках сессии, чтобы при повторном запросе предлагать новые варианты.

Поддерживаются два метода авторизации: официальная через Steam OpenID с безопасной передачей данных и автоматическим получением аватара с никнеймом, либо по прямой ссылке на профиль Steam с поддержкой как SteamID64, так и vanity URL (требует открытый профиль в настройках приватности).

Интерфейс адаптирован под разные устройства. Desktop версия использует полноразмерные карточки игр с hover эффектами и расширенной информацией. Mobile версия имеет компактный режим отображения (3 игры в ряд), оптимизированную навигацию и поддержку жестов с адаптированными элементами управления.

## Безопасность

CORS настроен для конкретных доменов (localhost и продакшен домен). Реализовано ограничение AI запросов - 10 запросов в минуту на IP адрес. Выполняется проверка входных параметров с ограничением на количество обрабатываемых игр (максимум 5000) и санитизацией пользовательского ввода. Все ошибки и важные события записываются в app.log с разделением на уровни INFO, WARNING и ERROR.

## Известные ограничения

Steam API может временно блокировать запросы при превышении лимитов. Wishlist интеграция недоступна через публичный API. Некоторые игры могут иметь региональные ограничения по ценам. Для просмотра библиотеки требуется открытый профиль Steam.

## Установка и запуск

Клонируйте репозиторий и перейдите в директорию проекта:

```bash
git clone https://github.com/dodry-question/steam-library.git
cd steam-library
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Создайте файл `.env` в корне проекта с необходимыми ключами:

```env
STEAM_API_KEY=ваш_ключ_steam
GROQ_API_KEY=ваш_ключ_groq
VSEGPT_BASE_URL=https://api.vsegpt.ru/v1/chat/completions
VSEGPT_API_KEY=ваш_ключ_vsegpt
MY_DOMAIN=http://localhost:8000
```

Получить API ключи можно на следующих ресурсах: Steam API Key на https://steamcommunity.com/dev/apikey, Groq API Key на https://console.groq.com/, VseGPT API Key на https://vsegpt.ru/

Запустите приложение:

```bash
python main.py
```

Приложение будет доступно по адресу http://localhost:8000

## Деплой на продакшен

Для локального запуска достаточно выполнить `python main.py`. Для деплоя на Amvera нужно создать проект на https://amvera.ru, подключить Git репозиторий, добавить переменные окружения в панели управления и обновить CORS в main.py для вашего домена.

Переменные окружения для продакшена:

```env
STEAM_API_KEY=ваш_ключ
GROQ_API_KEY=ваш_ключ
VSEGPT_API_KEY=ваш_ключ
VSEGPT_BASE_URL=https://api.vsegpt.ru/v1/chat/completions
MY_DOMAIN=https://your-app.amvera.io
```
