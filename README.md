# Steam Library Manager

Веб-приложение для управления и анализа библиотеки Steam с персонализированными рекомендациями игр на основе искусственного интеллекта.

## Возможности

- Просмотр библиотеки Steam с актуальными ценами и скидками
- Персонализированные рекомендации игр на основе анализа профиля
- Поиск и сортировка игр по различным параметрам
- Адаптивный дизайн для desktop и mobile устройств
- Просмотр библиотек других пользователей
- Статистика времени в играх
- Переключение между темной и светлой темой
- Кэширование данных для быстрой загрузки

## Технологический стек

**Backend:**
- FastAPI - асинхронный веб-фреймворк
- SQLModel - ORM для работы с базой данных
- httpx - асинхронные HTTP запросы
- SQLite - локальная база данных

**Frontend:**
- Vanilla JavaScript - модульная архитектура
- CSS3 - адаптивный дизайн с поддержкой тем
- Jinja2 - шаблонизатор

**Внешние API:**
- Steam Web API - получение данных о библиотеке
- Steam Store API - информация о ценах и жанрах
- VseGPT API (GPT-4o-mini) - генерация рекомендаций

## Архитектура и принцип работы

### Система кэширования

Приложение использует SQLite для кэширования данных о играх:
- При первом запросе данные загружаются из Steam API
- Информация сохраняется в локальную базу данных
- При повторных запросах данные берутся из кэша
- Кэш обновляется автоматически с учетом времени обновления Steam Store (10:10 PT)

### Фоновая загрузка данных

Реализована система потоковой загрузки (Streaming Response):
1. Сначала отображается список игр с базовой информацией
2. Детальные данные (цены, жанры, скидки) загружаются в фоновом режиме
3. Карточки игр обновляются по мере получения данных
4. Используется rate limiting для предотвращения блокировки Steam API

### Система рекомендаций

Рекомендации генерируются с помощью языковой модели GPT-4o-mini:

**Глобальный режим:**
- Анализируются топ-15 игр по времени в игре
- Учитывается выбранное настроение (жанровые предпочтения)
- Исключаются игры, уже имеющиеся в библиотеке

**Режим выделения:**
- Пользователь выбирает конкретные игры (Shift+клик или долгое нажатие)
- Модель ищет похожие игры по геймплею и атмосфере
- Результаты отображаются с пояснениями

**Защита от повторов:**
- История рекомендаций сохраняется в рамках сессии
- При повторном запросе модель предлагает новые варианты
- Отдельная история для каждого настроения

### Авторизация

Поддерживаются два метода входа:

**Steam OpenID:**
- Официальная авторизация через Steam
- Безопасная передача данных
- Автоматическое получение аватара и никнейма

**По ссылке профиля:**
- Ввод прямой ссылки на профиль Steam
- Поддержка как SteamID64, так и vanity URL
- Требует открытый профиль в настройках приватности

### Адаптивный интерфейс

**Desktop версия:**
- Полноразмерные карточки игр
- Hover эффекты и анимации
- Расширенная информация о играх

**Mobile версия:**
- Компактный режим отображения (3 игры в ряд)
- Оптимизированная навигация
- Поддержка жестов (долгое нажатие для выделения)
- Адаптированные элементы управления

## Безопасность

### CORS

Настроены ограничения для конкретных доменов:
```python
allow_origins=[
    "https://your-domain.com",
    "http://localhost:8000",
    "http://localhost:8001"
]
```

### Rate Limiting

Ограничение AI запросов: 10 запросов в минуту на IP адрес.

Настройка в `main.py`:
```python
AI_RATE_LIMIT = 10  # запросов
AI_RATE_WINDOW = 60  # секунд
```

### Валидация данных

- Проверка входных параметров
- Ограничение на количество обрабатываемых игр (максимум 5000)
- Санитизация пользовательского ввода

### Логирование

Все ошибки и важные события записываются в `app.log`:
- INFO - общая информация о работе приложения
- WARNING - предупреждения о потенциальных проблемах
- ERROR - критические ошибки с полным stack trace

## Известные ограничения

- Steam API может временно блокировать запросы при превышении лимитов
- Wishlist интеграция недоступна через публичный API
- Некоторые игры могут иметь региональные ограничения по ценам
- Требуется открытый профиль Steam для просмотра библиотеки

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/dodry-question/steam-library.git
cd steam-library
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
STEAM_API_KEY=ваш_ключ_steam
GROQ_API_KEY=ваш_ключ_groq
VSEGPT_BASE_URL=https://api.vsegpt.ru/v1/chat/completions
VSEGPT_API_KEY=ваш_ключ_vsegpt
MY_DOMAIN=http://localhost:8000
```

**Получение API ключей:**
- Steam API Key: https://steamcommunity.com/dev/apikey
- Groq API Key: https://console.groq.com/
- VseGPT API Key: https://vsegpt.ru/

### 4. Запуск приложения

```bash
python main.py
```

Приложение будет доступно по адресу: http://localhost:8000
